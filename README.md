# Collins Creek Farms

Website for Collins Creek Farms, a family farm in Cumming, Georgia.

## Hosting
This site is hosted using GitHub Pages and served at:
https://collinscreekfarms.com

## Structure
- `index.html` – Home page
- `about.html` – About the farm
- `contact.html` – Contact information
- `assets/` – CSS, images, and JavaScript

## Making Updates
1. Edit the HTML or CSS files
2. Commit and push to the `main` branch
3. GitHub Pages will automatically redeploy

## Notes
This is a static site with no backend or framework dependencies.
